PROJECTS

Game "Find all pair pictures"

This game was created using Windows Forms and C/C++. The idea is to find all pair pictures on the selected topic during the fixed time. Right after the user has found all the pair pictures, he recieves bonuses and goes to the next level, otherwise he is defeated. The game implements role-based authorization; Microsoft SQL Server Database is connected too

Tools and technologies: C/C++, Windows Forms, MS SQL Server

Web-app "Generator Timetable"

Web application built using the Python framework Django, designed to automatically generate school schedules. The application employs a genetic algorithm to create the most optimal schedule based on given norms, rules, and requirements. Additionally, it verifies that the generated schedule complies with the set norms and requirements. The application features an adaptive, simple, and user-friendly interface. Future plans include the integration of AI tools to enhance its functionalities.

Tools and technologies: Python, Django, MySQL, HTML5, SCSS, JavaScript/jQuery
